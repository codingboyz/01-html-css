## HTML Colors
 - HTML 에서 Color를 세팅하는법
   - Text : Tomato, Orange, Blue와 같이 이미 색이 명시된 글자로 세팅
   - RGB : ```rgb(255,255,255)``` 와 같이 나타냄 ```rgb(red, green, blue)``` 순이다
   - HEX : #000000 과 같이 나타냄
   
 - Color Picker나 Adobe CC에서 좋은 색찾아쓰자
